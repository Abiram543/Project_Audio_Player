# Project_Audio_Player
Audio_Player Project

Design an audio playback system capable of playing .wav files through the basic  audio output circuitry available on the Blackboard platform. 
Initially, a small audio file will be  preloaded into a Block RAM, and your playback circuit will sequentially read audio samples from  memory and transfer them to a PWM-based digital-to-analogue converter (DAC) for playback.
